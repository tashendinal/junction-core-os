# junction-core-os
Junction Core OS: A modular, distributed broadcast OS for Orange Pi 6 Ultra clusters. Features multi camera NDI 6.0 ingest, 45 TOPS AI color matching (Sony/Nikon/Canon/red), PTP timecode sync, and DaVinci Resolve live-grade integration. Optimized for 1080p 24p cinema production with SLT-Fiber remote rescue capability.
